# spiderlein_allowlist.txt
#
# Controls which modules appear in the spiderlein output (--mode spiderlein).
# Format:  category/software   (lowercase)
# Entries use DISPLAY names, i.e. names AFTER rename mappings from
# spiderlein_cat_rename.json have been applied.
# Example: gcc is renamed to gnu → use "compiler/gnu" here.
#
# Only exact category/software pairs are matched — i.e. "compiler/gnu" includes
# all versions of GCC, but NOT gcccore, go, etc.
#
# If this file is empty (contains only comments / blank lines), ALL modules
# are written to the output without filtering.
